1. ifconfig
   kaam: Apna IP address dekhna
   Example: ifconfig

2. ping
   kaam: Server se connection check karna, latency dekhna
   Example: ping google.com

3. curl ifconfig.me
   kaam: Apna Public IP dekhna
   Example: curl ifconfig.me

4. nslookup
   kaam: Domain ka IP address pata karna (dNS)
   Example: nslookup google.com



5. nslookup
   Kaam: Domain ka IP address pata karna (DNS)
   Example: nslookup google.com

6. netstat -tulpn
   Kaam: Active network connections aur ports dekhna
   Example: netstat -tulpn

7. traceroute
   Kaam: Data ka path dekhna server tak
   Example: traceroute google.com

8. ssh
   Kaam: Remote server se secure connect karna
   Example: ssh username@server_ip
